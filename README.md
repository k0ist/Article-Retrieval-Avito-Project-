# Гибридный RAG поиск — двухэтапный поиск релевантных статей (MAP@10)

Решение задачи поиска релевантных статей справки по пользовательскому запросу.
Для каждого запроса возвращается ранжированный список из до 10 `article_id`,
отсортированный по убыванию релевантности.

## Архитектура

Пайплайн состоит из двух этапов:

**1. Retrieval (широкий поиск кандидатов)**
- **BM25** — лексический поиск по стеммированным токенам (`nltk` +
  `SnowballStemmer("russian")`), работает по полю `title` (с утроенным весом) + `body`.
- **Dense embeddings** — семантический поиск через `intfloat/multilingual-e5-large`
  и `faiss` (`IndexFlatIP`), работает по укороченному тексту (заголовок + первые
  150 слов текста статьи).
- Результаты BM25 и embeddings объединяются через **Reciprocal Rank Fusion (RRF)**.

**2. Reranking (точное ранжирование для улучшения MAP@10)**
- Top-50 кандидатов после RRF пересортировываются **cross-encoder**'ом
  (`BAAI/bge-reranker-v2-m3`), который оценивает пару (запрос, текст статьи)
  целиком, а не по отдельным векторам — это точнее bi-encoder подхода,
  но дороже по вычислениям, поэтому применяется только к узкому пулу кандидатов.
- Из пересортированного списка берутся top-10.

## Структура данных

| Файл | Описание |
|---|---|
| `articles.f` | база статей: `article_id`, `title`, `body` (HTML) |
| `calibration.f` | 500 размеченных запросов для локальной проверки: `query_id`, `query_text`, `ground_truth` |
| `test.f` | 500 запросов без разметки: `query_id`, `query_text` |
| `answer.csv` | результат: `query_id`, `answer` (id статей через пробел, топ-10, порядок важен) |

## Метрика

Решение оценивается по **MAP@10** — среднему значению AP@10 по всем запросам,
где чем выше в списке релевантный документ, тем больше его вклад.

```bash
python main.py
```

## Скрипт:
1. Строит корпус статей (очистка HTML, стемминг, отдельные тексты для BM25 и embeddings).
2. Считает `MAP@10` на `calibration.f` и печатает в консоль.
3. Генерирует предсказания для `test.f` и сохраняет `answer.csv`.

## Используемые модели

| Компонент | Модель |
|---|---|
| Эмбеддинги | `intfloat/multilingual-e5-large` |
| Reranker | `BAAI/bge-reranker-v2-m3` |
